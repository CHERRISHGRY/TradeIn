# 二手交易平台
我的毕业设计项目，二手交易平台，dagger2+mvp+Bmob后台云搭建，集成了百度地图，友盟三方登录等
##系统架构
Dagger2+MVP分层，完成了一次正常的retrofit下的天气信息的请求，其余请求后台均基于Bmob云后台，图片在水平方向可滚动
##说明
使用请尊重本人技术，有好的架构修改意见请提issue。本人CSDN博客[CSDN](http://blog.csdn.net/u010132993)
##使用技术
Dagger2
MVP
Retrofit2
Okhttp3
Butterknife
rxjava
Bmob
